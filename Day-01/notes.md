# Day 1 - Two Sum Problem

## What I Learned
- Used hash maps in C++
- Learned optimized approach for Two Sum
- Improved understanding of time complexity

## Approach
1. Traverse the array
2. Find complement = target - current element
3. Check if complement already exists in hashmap
4. If yes, return indices

## Time Complexity
O(n)

## Space Complexity
O(n)

## Concepts Used
- Arrays
- Hash Map (`unordered_map`)
- Searching
